# Ресурсы

## Основная книга

- Р. Гонсалес, Р. Вудс «Цифровая обработка изображений».
  Важно: Обязательно решай задачи в конце каждой главы (раздел «Задачи»).

## Задачники по математике (по мере необходимости)

- https://immersivemath.com/ila/?spm=a2ty_o01.29997173.0.0.7af35171LFdYHy
- https://ocw.mit.edu/courses/18-06sc-linear-algebra-fall-2011/
- Линейная алгебра: С. Липшутц «Линейная алгебра» (операции с матрицами).
- Теория вероятностей: В.Е. Гмурман «Руководство к решению задач...» (шум, гистограммы).

- Библиотеки (на будущее): p5.js (прототипы), three.js / ogl (WebGL), lil-gui (интерфейс для инсталляций).
