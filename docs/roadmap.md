# Roadmap

## Модуль 0: Подготовка (1 неделя)

- Цель: Настроить среду, понять память изображения.
- Главы: 1.1–1.5, 2.1–2.4.
- Математика:
  - [ ] Системы счисления (бинарная, HEX).
  - [ ] Объем памяти: расчет размера файла для разных разрешений.
  - [ ] Индексация массивов (перевод (x, y) в индекс 1D массива).
- Практика TS:
  - [ ] Настроить проект, убрать линтеры-ошибки.
  - [ ] Написать функции getPixel(x, y) и setPixel(x, y).
  - [ ] Реализовать конвертацию в оттенки серого (формула яркости).
  - [ ] Сделать экспорт результата в PNG.

## Модуль 1: Точечные операции и Гистограммы (2–3 недели)

- Цель: Управление контрастом и тоном.
- Главы: 3.1–3.3.
- Математика:
  - [ ] Построить гистограмму вручную для массива из 20 чисел.
  - [ ] Рассчитать преобразование яркости (негатив, гамма) для 5 значений вручную.
  - [ ] Рассчитать функцию эквализации гистограммы для маленького примера.
- Практика TS:
  - [ ] Фильтры: Негатив, Гамма-коррекция, Логарифмическое сжатие.
  - [ ] Построение визуализации гистограммы на отдельном canvas.
  - [ ] Алгоритм: Эквализация гистограммы (Histogram Equalization).
  - [ ] Арт-задача: Создать генеративный паттерн, где цвет зависит от эквализации шума.

## Модуль 2: Пространственная фильтрация (3–4 недели)

- Цель: Понять свертку (основа всей графики).
- Главы: 3.4–3.7.
- Математика:
  - [ ] Критично: Вручную выполнить свертку матрицы 5x5 с ядром 3x3.
  - [ ] Рассчитать градиент (оператор Собеля) для фрагмента 3x3.
  - [ ] Изучить типы padding (zero, replicate, reflect).
- Практика TS:
  - [ ] Написать универсальную функцию convolve(image, kernel).
  - [ ] Реализовать фильтры: Размытие (Box, Gaussian), Медианный (шумодав).
  - [ ] Реализовать фильтры резкости (Лапласиан, Unsharp Mask).
  - [ ] Арт-задача: Эффект «масляной живописи» через нелинейные фильтры.

## Модуль 3: Частотная область и переход на GPU (4–5 недель)

- Цель: Понять Фурье и ускорить графику для проекций.
- Главы: 4.1–4.11.
- Математика:
  - [ ] Расчет ДПФ (Discrete Fourier Transform) для последовательности из 4–8 чисел вручную.
  - [ ] Понять свойства симметрии спектра.
  - [ ] Нарисовать АЧХ фильтров (Low-pass, High-pass).
- Практика TS (CPU → GPU):
  - [ ] Реализовать 1D ДПФ кодом (для понимания).
  - [ ] Для 2D использовать библиотеку (например, fourier-transform), визуализировать спектр (сдвиг нуля в центр, логарифмирование).
  - [ ] Переход на WebGL: Изучить базовые шейдеры (vertex/fragment).
  - [ ] Переписать фильтры (размытие, края) на GLSL шейдеры.
  - [ ] Арт-задача: Генеративная анимация шума в частотной области (FFT noise).

## Модуль 4: Цвет и Восстановление (3 недели)

- Цель: Работа с цветом и шумом.
  Главы: 5.1–5.4, 6.1–6.7.
- Математика:
  - [ ] Генерация случайных чисел (Гаусс, Равномерное) вручную/теория.
  - [ ] Расчет метрик качества (MSE, PSNR) для двух матриц.
  - [ ] Конвертация формул RGB ↔ HSI.
- Практика TS:
  - [ ] Программное добавление шума (Гаусс, Соль-перец).
  - [ ] Удаление периодического шума (через частотную область).
  - [ ] Конвертация цветовых пространств.
  - [ ] Арт-задача: Сегментация изображения по цвету (выделение объекта) для интерактива.

## Модуль 5: Морфология и Сегментация (3 недели)

- Цель: Работа с формами и бинарными образами.
- Главы: 9.1–9.6, 10.1–10.3.
- Математика:
  - [ ] Выполнить эрозию/дилатацию на бинарной матрице 5x5 вручную.
  - [ ] Разобрать алгоритм Оцу (формула дисперсии).
- Практика TS:
  - [ ] Морфологические операции (Эрозия, Дилатация, Открытие, Закрытие).
  - [ ] Алгоритм Оцу для автоматической бинаризации.
  - [ ] Арт-задача: Стилизация видео/изображения под «гравюру» или «силуэт» в реальном времени.

## Модуль 6: Финальный Проект (Постоянно)

- Цель: Инсталляция для проектора.
- Задачи:
  - [ ] Выбрать тему (например, «Живой портрет» или «Абстрактный поток»).
  - [ ] Объединить изученные фильтры в пайплайн.
  - [ ] Добавить интерфейс (lil-gui) для настройки параметров на лету.
  - [ ] Оптимизировать под 60 FPS (через WebGL).
  - [ ] Протестировать на целевом разрешении проектора (например, 1920x1080 или 4K).
